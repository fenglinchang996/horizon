## Usage Suggestion

1. Start with `Grammar` and `Types and Objects` for language and type fundamentals.
2. Continue with `Functions and Scope` to build a solid execution mental model.
3. Practice `Arrays` for common APIs, iteration patterns, and pitfalls.
4. Learn `Async` for Promise and event loop behavior.
5. Move on to `Logic and Algorithms` and `Prototypes and Inheritance` for advanced interviews.
